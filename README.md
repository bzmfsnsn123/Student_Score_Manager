# Student Score Manager
## 学生成绩控制台管理系统（Python练手项目）
纯控制台实现学生成绩全流程管理，支持本地文件持久化存储，重启程序数据不丢失。覆盖字典、文件读写、循环菜单、数值统计、异常捕获等Python基础核心知识点，适合新手巩固语法。

## 项目功能
1. **新增学生成绩**：录入姓名与0-100分有效分数，重复姓名拦截校验，自动保存到本地文本
2. **单人成绩查询**：输入姓名快速调取对应分数，无匹配学生给出提示
3. **全班数据统计**：展示全部学生名单，自动计算平均分、最高分、最低分、总人数
4. **修改已有成绩**：选中学生更新分数，数据实时写入本地文件
5. **删除学生记录**：一键移除指定学生整条成绩数据
6. **完善输入容错**：非数字、空白姓名、超范围分数、无效序号均友好提示，程序不会崩溃退出
7. **数据持久化**：所有成绩存储在 `score.txt`，关闭程序数据不丢失，再次打开自动加载

## 运行说明
### 环境要求
Python 3.6 及以上版本，无第三方依赖库，系统内置语法直接运行。

### 使用步骤
1. 将源码保存为 `score_manager.py`，同目录下会自动生成 `score.txt` 存储成绩
2. 使用 PyCharm / IDLE / 终端执行脚本，进入功能菜单
3. 菜单指令说明：
    - 1：添加新学生成绩
    - 2：查询单个学生分数
    - 3：查看全班成绩与分数统计
    - 4：修改现有学生分数
    - 5：删除学生记录
    - 0：保存全部数据并退出程序

## 核心学习知识点
1. **字典 Dict**：以学生姓名为键、分数为值存储数据，快速增删改查
2. **文件持久化读写**：`open()`、`with` 上下文管理器，文本分行存储、逗号分割解析数据
3. **无限菜单循环**：`while True` 常驻主界面，`break` 控制程序退出
4. **字典遍历 `dict.items()`**：批量读取全部学生数据打印列表
5. **集合统计函数**：`sum()` / `max()` / `min()` / `len()` 批量计算分数指标
6. **数值格式化**：`:.1f` 控制浮点数保留1位小数
7. **异常捕获 try-except**：拦截非法数字输入，提升程序健壮性
8. **字典操作**：新增赋值、`del` 删除键值对、键存在性判断
